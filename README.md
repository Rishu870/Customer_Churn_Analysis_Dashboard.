# Customer_Churn_Analysis_Dashboard.
Developed a Customer Churn Analysis dashboard using SQL and Power BI to identify key factors influencing customer attrition. Performed data cleaning, created calculated fields, and built interactive visualizations to analyze churn based on contract type, tenure, monthly charges, and customer segments. The dashboard provides actionable insights .


1. Problem Statement

“The telecom company is facing customer churn, which directly impacts revenue and customer lifetime value. The objective of this project is to analyze customer data and identify key factors contributing to churn, so that the company can take proactive steps to retain customers.”

🛠️ 2. Solution Approach

“To solve this problem, I performed data analysis using SQL and built an interactive Power BI dashboard. I cleaned the data, created a churn flag column, and analyzed customer behavior based on contract type, payment method, tenure, and monthly charges. The dashboard provides a clear visualization of patterns and helps identify high-risk customers.”

 3. What Your Dashboard Shows (Insights 🔥)
 1. Contract Type Drives Churn
Customers with month-to-month contracts have the highest churn
 Meaning: Lack of long-term commitment increases churn
 2. High Monthly Charges → More Churn
Customers paying higher charges are more likely to leave
 Meaning: Pricing strategy needs optimization
 3. Low Tenure Customers Churn More
Customers with short tenure (new customers) churn faster
 Meaning: Onboarding & early engagement is weak
 4. Senior Citizen Analysis
Majority of customers are non-senior citizens
But churn distribution still needs attention
 Meaning: Different customer segments behave differently
 4. Business Recommendations

Based on the dashboard insights:

 Offer discounts for high monthly charge customers
 Promote long-term contracts (1-year / 2-year plans)
 Improve early customer engagement (first 3–6 months)
 Provide better support & retention strategies
